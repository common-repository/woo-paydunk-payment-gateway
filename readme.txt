== Description ==

This plugin enables paydunk payment gateway installation easy for woocommmerce. The plugin has all the settings required to setup paydunk gateway working for the store.

Paydunk payment gateway is the new way of getting paid in the store where users' don't have to fill in all the information e.g. shipping address, billing address name etc. User will download the app, register mobile number and fill in billing address shipping address in the app including credit card. Once user initiate the payment on the store, user will receive push notification, on the registered mobile number. Once user confirm the payment using pin, the payment will be processed through selected payment gateway.

Currently this plugin supports PayPal and Authorize.net as payment gateways.

More information at http://www.paydunk.com/ and http://developers.paydunk.com/
